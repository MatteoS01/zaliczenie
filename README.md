## O projekcie
Projekt powstał jako zaliczenie przedmiotu *Zarządzanie projektem wdrożeniowym*. Założeniem było stworzenie prostej, w pełni statycznej witryny prezentującej informacje o zwierzętach – bez backendu i bazy danych, opartej wyłącznie o front-end.

## Funkcjonalności
- karty z opisami poszczególnych gatunków zwierząt
- responsywny układ strony (RWD)
- przejrzysta, intuicyjna nawigacja
